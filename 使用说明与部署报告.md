# A 股做 T 信号监控工具使用说明与部署报告

更新时间：2026-06-03

## 一、这个软件是做什么的

`AStock T Signal Monitor` 是一个个人 A 股盘中做 T 辅助提醒工具。它会在 A 股交易时间内拉取分钟行情，按模型判断自选股是否出现日内做 T 机会；如果触发信号，会在软件界面显示，并通过微信推送给用户。

它不是自动交易软件，不会替你下单，也不构成投资建议。真实交易仍需要你自己结合盘口、仓位、流动性和风险承受能力判断。

当前版本重点功能：

- 可视化 Windows EXE，不需要用户自己装 Python。
- 支持 macOS 版本。
- 支持 PushPlus 推送。
- 支持内置微信扫码登录推送，推荐填 `weixin`。
- 支持自选监控池：只推送自选池里的做 T 信号。
- 支持盘中信息异动、小作文雷达、大盘/自选板块提醒。
- 支持自选走势、分时/日线/周线/月线、支撑压力等辅助观察。
- 支持 AI 研报窗口和信息面搜索。
- 支持点击「更新程序」拉取新版。
- 主界面已改成可滚动页面，小窗口也能看到下面的按钮、表格和日志。

## 三、第一次打开怎么用

打开软件后主要看这几个输入框：

```text
模型文件/文件夹
PushPlus token
微信推送
自选监控池
间隔秒
```

推荐设置：

```text
模型文件/文件夹：默认 models 文件夹即可
PushPlus token：可以留空，如果你只用内置微信推送
微信推送：weixin
自选监控池：填你想监控的股票，比如 锡业股份，东山精密
间隔秒：30
```

然后按顺序操作：

1. 在「微信推送」里填 `weixin`。
2. 点击「微信登录」。
3. 扫码登录。
4. 在微信里给 bot 发一句话，例如 `你好`。
5. 回到软件点「刷新会话」。
6. 点「测试推送」，确认手机微信能收到消息。
7. 点「开始监控」。

如果你想用 PushPlus：

1. 打开 PushPlus 官网。
2. 用微信扫码登录。
3. 进入「一对一推送」页面。
4. 复制 token。
5. 粘贴到软件的 `PushPlus token` 输入框。
6. 点「测试推送」。
7. 手机微信收到测试消息后再点「开始监控」。

PushPlus 官网：

```text
https://www.pushplus.plus/
```

## 四、自选监控池怎么理解

自选监控池是当前版本很重要的入口。

例如你填：

```text
锡业股份，东山精密
```

那么：

- 做 T 信号只检查这两只已经有模型的股票。
- 微信只推送这两只的做 T 信号。
- 盘中信息异动只围绕这两只扫描。
- 小作文雷达只围绕这两只扫描。
- 大盘/自选板块提醒会按这两只的概念和篮子生成监控。

如果你清空自选监控池，软件会默认加载模型文件夹里的全部模型。

当前内置模型包括：

```text
剑桥科技 603083.XSHG
东山精密 002384.SZ
福晶科技 002222.SZ
利通电子 603629.XSHG
锡业股份 000960.SZ
```

默认自选池：

```text
剑桥科技，东山精密，福晶科技，利通电子，锡业股份
```

## 五、做 T 信号怎么看

当软件检测到做 T 信号，会显示并推送类似内容：

```text
股票：锡业股份
动作：建议买入T仓 / 建议卖出T仓
入场价：43.83
目标1/2/3：44.18 / 44.53 / 44.88
参考风控价：43.30
评分：2.5
大盘涨跌、篮子涨跌、相对强弱、量能比等
```

### 正 T

```text
建议买入T仓
目标卖出
```

意思是先买入一笔 T 仓，后面如果冲到目标价，考虑卖出相同数量，赚日内价差。

### 倒 T

```text
建议卖出T仓
目标买回
```

意思是先卖出已有底仓的一部分，后面如果跌到目标价，考虑买回相同数量，降低持仓成本。

### 目标1/2/3

当前推送会给三档目标价：

```text
目标1：半档目标，偏保守
目标2：模型主目标
目标3：强势延伸目标
```

实际使用时可以：

- 目标1 先减一部分。
- 目标2 做主要成交计划。
- 目标3 只在强势行情里挂远一点。

## 六、当前策略逻辑

当前版本是激进日内做 T 口径：

```text
只评估信号后当天最大可捕捉价差
不评估底仓隔夜走势
正T/倒T均可
一天可给多个点位
```

主要因子包括：

- 目标股分时走势。
- 相似股篮子走势。
- 大盘走势：上证、深成、创业板加权。
- 目标股相对篮子强弱。
- 分时均价偏离。
- 篮子分化程度。
- 量能比。

当前每只股票每天最多推送：

```text
5 个做T信号
```

同方向信号之间至少间隔：

```text
10 分钟
```

注意：这里的回测胜率是「信号后当天曾经给到可做 T 价差」的胜率，不等于实际成交胜率。真实结果会受挂单、滑点、盘口深度、手动执行速度影响。

回测窗口：近 30 个交易日分钟级数据。

## 八、盘中信息异动和小作文雷达

如果勾选：

```text
盘中信息异动
```

软件会在交易时间内定期扫描自选股的信息面。如果出现情绪极差、传播性强、可能影响股价的事件簇，会推送到微信。

它不是判断真假，而是判断：

- 传播热度。
- 杀伤力。
- 可信度。
- 是否已有官方澄清。
- 是否与行情跳水/放量共振。

小作文雷达适合用来捕捉不一定来自正式媒体、但可能已经影响盘面的情绪事件。

手动扫描时，软件只展示结果，不推送，也不计入自动推送去重。

## 九、大盘/自选板块提醒

如果勾选：

```text
大盘/自选板块
```

软件会监控：

- 上证指数。
- 深成指。
- 创业板指。
- 自选股相关概念板块。
- 自选股相似股篮子。

当前逻辑尽量避免一天重复推太多条，只在首次变化、幅度扩大、状态反转时提醒。

## 十、常用按钮说明

### 模型盘前

根据模型回测摘要和当前模型参数，生成今日重点观察顺序。

### 信息面盘前

汇总自选股盘前信息面，适合开盘前看有没有明显利好/利空。

### 自选信息面

手动搜索某只股票或关键词的信息面。

### 小作文雷达

手动扫描最近若干小时的小作文/传闻/负面情绪事件。

### 自选走势

查看自选股走势，包含分时、日线、周线、月线等，并辅助标注支撑压力。

### AI研报

按股票生成偏研究报告风格的摘要，包含市场空间、同行、估值、风险等。

### 更新程序

点击后会检查 GitHub 最新版本。如果发现新版本，会下载更新包并提示打开新版。

### 测试推送

用于确认 PushPlus 或内置微信推送是否接通。

## 十一、常见问题

### 1. 为什么收盘后还有界面显示信号？

软件收盘后应该只显示最后行情，不再记录和推送新信号。如果你看到旧信号，多半是表格里保留了上一次监控结果；以推送和日志时间为准。

### 2. 为什么我填了自选池，还是看不到某只股票？

做 T 信号只会监控「有模型 JSON」的股票。比如你填一个没有模型的股票，信息面可以搜索，但做 T 信号不会出现。

解决方法：

- 使用已有模型股票。
- 或新增该股票模型 JSON。
- 或让程序重新训练该股票模型。

### 3. 为什么 token 无效？

PushPlus token 必须来自你自己的 PushPlus 账号「一对一推送」页面。朋友的 token 只能推给朋友自己的微信。

如果用内置微信推送，可以不填 PushPlus token，只在微信推送里填：

```text
weixin
```

### 4. 为什么 Windows 弹出临时目录删除失败？

这是 PyInstaller 单文件 EXE 偶发问题，通常是程序退出时临时目录还被占用。一般不影响使用。可以关闭软件后稍等，或者重启电脑后再打开。

### 5. 为什么 macOS 打开闪退？

优先用包里的 `RUN_FROM_TERMINAL.command` 启动，看日志。也可以解除隔离：

```bash
xattr -dr com.apple.quarantine AShareTSignalMonitor.app
```

### 6. 为什么新闻接口 SSL 失败？

如果报：

```text
SSL: CERTIFICATE_VERIFY_FAILED
```

通常是证书链问题。新版构建已经尽量打包 `certifi`，如果仍出现，需要更新到最新 EXE，或把错误日志发出来定位。

## 十二、模型和自动更新

模型文件在：

```text
models/*.json
```

每个 JSON 对应一只股票，包含：

- 股票名。
- 股票代码。
- 相似股篮子。
- 做 T 参数。
- 回测摘要。

GitHub Actions 会自动构建新版 Windows EXE 和 macOS 包。你点击「更新程序」后，软件会尝试拉取最新版本。

当前 release 下载地址固定：

```text
macOS:   https://github.com/Roypic/Astock_t/releases/download/macos-latest/AShareTSignalMonitor-macOS.zip
```

## 十三、风险说明

这个工具只做辅助提醒：

- 不保证数据实时性。
- 不保证信号一定成交。
- 不保证历史回测未来继续有效。
- 不考虑你的真实持仓成本。
- 不知道你的仓位、手续费、滑点和交易纪律。
- 不会自动下单。

做 T 尤其要注意：

- 追高买入可能当天无法卖出。
- 倒 T 卖出后可能买不回。
- 小作文和情绪事件可能真假混杂。
- 板块急跌时，模型信号可能失效。

建议把它当作「提醒器」和「盘中观察助手」，不要当成自动赚钱系统。
